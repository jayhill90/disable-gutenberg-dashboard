Disable the try Gutenberg dashboard widget from showing in WordPress 4.9.

== Description ==

The plugin will stop the try Gutenberg dashboard widget from showing on sites running WordPress 4.9.

== Frequently Asked Questions ==

= What does the plugin do? =

It will stop the try Gutenberg dashboard widget from showing on any sites running WordPress 4.9.

== Installation ==

1. Upload the `disable-gutenberg-dashboard` directory into `wp-content/plugins/`
2. Activate the plugin through the "Plugins" menu in WordPress
